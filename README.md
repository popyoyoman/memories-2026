# 2026 珍貴回憶錄 📸

這是一個為了保存與回顧 2026 年珍貴瞬間而建立的數位回憶專案。

## 🌐 專案網址
* **珍貴回顧相簿**: [https://popyoyoman.github.io/memories-2026/](https://popyoyoman.github.io/memories-2026/)
* **回憶拼圖小遊戲**: [https://popyoyoman.github.io/memories-2026/puzzle.html](https://popyoyoman.github.io/memories-2026/puzzle.html)

## ✨ 主要功能

### 1. 珍貴回顧相簿 (`index.html`)
* **海量照片呈現**: 自動化處理並展示 667 張珍貴生活照片。
* **流暢體驗**: 採用 Masonry 網格佈局，支援無限捲動 (Infinite Scroll) 與延遲載入 (Lazy Loading)。
* **溫馨氛圍**: 內建隱藏式背景音樂播放器（經典鋼琴曲），可透過右下角控制鈕自由開啟。
* **全螢幕瀏覽**: 點擊照片即可開啟燈箱 (Lightbox) 模式，細細品味每個瞬間。

### 2. 回憶拼圖小遊戲 (`puzzle.html`)
* **互動體驗**: 將 2026 年的照片隨機轉換為拼圖遊戲。
* **難度調整**: 支援 3x3 (簡單)、4x4 (中等)、5x5 (挑戰) 三種難度，適合全家人（包括長輩與小孩）一起玩。
* **輔助功能**: 提供「按住看提示」功能，方便在卡關時對照原圖。
* **過關慶祝**: 完成拼圖後會有動態慶祝特效 (Confetti)。

## 🛠️ 技術細節
* **前端**: HTML5, CSS3 (Modern Flexbox/Grid), JavaScript (ES6+)。
* **音樂**: YouTube IFrame Player API。
* **遊戲邏輯**: 基於滑塊拼圖 (Sliding Puzzle) 演算法，確保每次打亂皆有解。
* **部署**: GitHub Pages。
